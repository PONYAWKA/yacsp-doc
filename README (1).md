# Инструкция по запуску документации

Для локального запуска копии документации выполните следующие шаги:

1. Установите Python, если он не установлен.
2.  Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```
3.  Запустите локальный сервер:

    ```bash
    mkdocs serve
    ```
4. Откройте в браузере адрес `http://127.0.0.1:8000`.

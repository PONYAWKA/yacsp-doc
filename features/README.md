# Функционал YACS Panel

Основные функции YACS Panel, доступные каждому пользователю:

*   **Полностью автоматический фарм**
    Панель самостоятельно управляет аккаунтами: собирает лобби, ищет игру, совершает убийства. После получения уровня на всех аккаунтах YACS Panel автоматически выбирает награду, отправляет её на указанную трейд-ссылку и запускает следующую партию аккаунтов.
    *Панель 100% автоматизирована — фарм запускается нажатием пары кнопок.*

*   **Множество режимов**
    Поддержка фарма в **5х5** (Competitive) и **2х2** (Wingman).
    Панель может играть как в ничью, так и со случайным счётом.

*   **Фоновый фарм**
    Возможность настроить фарм в фоне, позволяя заниматься своими делами на ПК во время работы панели.
    *Не требует установки виртуальных машин (VM).*

*   **Реалистичный фарм (Simulation)**
    Полная симуляция игры: панель выполняет человекоподобные действия, такие как установка бомбы, разминирование и другие игровые сценарии.

*   **Модуль Activity Booster**
    Запуск различных игр на аккаунтах для накрутки часов и повышения "человечности" аккаунтов.

*   **Автоматический выбор дропа**
    Панель сама выбирает лучший дроп и отправляет его через трейд-оффер.

*   **Фарм Armory Pass**
    Поддержка режима для фарма пропусков (Armory Pass).

*   **Автоматизация питания**
    Функции автозапуска фарма при включении компьютера и автоматического выключения ПК после завершения работы всех аккаунтов.

*   **Подробная статистика**
    Отслеживание полученного дропа, уровней и XP аккаунтов.
    *   Отображение текущего уровня и XP в списке.
    *   Сортировка аккаунтов по уровню/XP.
    *   Индикация аккаунтов, уже отфармивших дроп на текущей неделе.

*   **Удобное управление**
    *   **Горячие клавиши**: Переключение между окнами аккаунтов (ALT + 1, ALT + 2 и т.д.).
    *   **Быстрый запуск**: Запуск аккаунтов с применением минимальных настроек графики и низкого разрешения.
    *   **Panic Button**: Закрытие всех аккаунтов одной кнопкой.

*   **Telegram уведомления**
    Уведомления о действиях панели, скриншоты и статистика наград прямо в мессенджер.

*   **Безопасность**
    Работа осуществляется **БЕЗ чтения памяти игры** и **БЕЗ инжекта** (внедрения) в процесс игры.

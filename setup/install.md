# Установка и настройка YACSPanel

Перед началом убедитесь, что вы выполнили все пункты из раздела [Установка необходимого ПО и настройка системы](software.md).

## Основные настройки панели для фарма

1. **Скачивание панели** Скачайте архив `YACS_Panel v.x.x.x.zip` последней версии из бота @yacsp\_bot (кнопка "Скачать панель"). Разархивируйте в корень диска `C:`. Например: `C:\YACSPanel`.
2. **Настройка антивируса** Добавьте папку с панелью в исключения **Windows Defender**, чтобы избежать блокировки или удаления файлов панели.
3. Запустите yacsp.exe
4. Введите ключ леценьзии полученный в боте

<figure><img src="../.gitbook/assets/image (2) (1).png" alt=""><figcaption></figcaption></figure>

1. **Настройка аккаунтов**
   *   **logpass.txt**: В папке с панелью откройте файл `logpass.txt` и добавьте свои аккаунты в формате:

       ```
       login:password
       login:password
       ```

       _Удалите тестовый аккаунт, если он там есть. Не забудьте сохранить файл._
   *   Нажмите кнопку Import Для импорта maFIles

       <figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>
2. **Путь к Steam** В панели нажмите **"Advanced Settigns"** и укажите путь к `steam.exe` .
   * Пример: `C:\Program Files (x86)\Steam\steam.exe`
   * _Если CTRL+V не работает, переключите раскладку на английскую._
3. **Путь к CS2** Нажмите **"CS2 path"** и укажите путь к папке `Counter-Strike Global Offensive`.
   * Пример: `C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive`

***

## Настройка уведомлений и управления (Telegram)

1. В панели перейдите на вкладку **"Advanced Settigns"** -> **"Telegram Token"**.
2. Откройте в Telegram бота `@BotFather`.
3. Отправьте команду `/newbot`, придумайте название и юзернейм (тег) для вашего бота.
4. Скопируйте полученный **API TOKEN**.
5. Вставьте токен в поле **"Telegram Token"**.
6. **ОБЯЗАТЕЛЬНО** найдите своего созданного бота в Telegram и нажмите `/start`.
7. В боте `@yacsp_bot` в главном меню скопируйте ваш **ID**.
8. Вставьте ID в поле **"TG Whitelist"**.
   * _Для уведомлений нескольким пользователям укажите ID через запятую: `12345,67890`._
9. Нажмите **"Save"**.
10. Нажмите "**Telegram Test**" - вы великолепны

> Через созданного бота можно управлять панелью командой `/start` (доступно только пользователям из Chat ID).

***

## Настройка Looter (Трейд-ссылки)

1. Перейдите на вкладку **"Advanced Settigns"** -> **"Trade URL"**.
2. Вставьте вашу трейд-ссылку (Trade URL) и нажмите **"Ok"**.
   * Пример: `https://steamcommunity.com/tradeoffer/new/?partner=...`

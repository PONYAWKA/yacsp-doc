# Установка и настройка YACS Panel

!!! info "Важно"
    Перед началом убедитесь, что вы выполнили все пункты из раздела [Установка необходимого ПО и настройка системы](software.md).

## Основные настройки панели для фарма

1.  **Скачивание панели**
    Скачайте архив `YACS_Panel v.x.x.x.zip` последней версии из бота `moonlighter_shop_bot` (кнопка "Скачать панель").
    Разархивируйте в корень диска `C:`. Например: `C:\YACSPanel`.

2.  **Настройка антивируса**
    Добавьте папку с панелью в исключения **Windows Defender**, чтобы избежать блокировки или удаления файлов панели.

3.  **Настройка аккаунтов**
    *   **logpass.txt**: В папке с панелью откройте файл `logpass.txt` и добавьте свои аккаунты в формате:
        ```text
        login:password
        login:password
        ```
        *Удалите тестовый аккаунт, если он там есть. Не забудьте сохранить файл.*
    *   **maFiles**: Скопируйте ваши `.maFile` файлы в папку `maFiles` внутри папки панели.
        > **ВАЖНО!** Мафайлы должны быть **БЕЗ ПАРОЛЯ**. Если в SDA установлен пароль шифрования, снимите его (кнопка "Manage Encryption").

4.  **Первый запуск**
    Запустите `Panel.exe`.

5.  **Введите Леценьзию**
    

6.  **Путь к Steam**
    В панели нажмите **"Set steam.exe path"** и укажите путь к `steam.exe` или папке Steam.
    *   Пример: `C:\Program Files (x86)\Steam\steam.exe`
    *   *Если CTRL+V не работает, переключите раскладку на английскую.*

7.  **Путь к CS2**
    Нажмите **"Set CS2 path"** и укажите путь к папке `Counter-Strike Global Offensive`.
    *   Пример: `C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive`

## Настройка уведомлений и управления (Telegram)

1.  В панели перейдите на вкладку **"Config #2"** -> **"Telegram notifications settings"**.
2.  Откройте в Telegram бота `@BotFather`.
3.  Отправьте команду `/newbot`, придумайте название и юзернейм (тег) для вашего бота.
4.  Скопируйте полученный **API TOKEN**.
5.  Вставьте токен в поле **"Telegram API TOKEN from @BotFather"**.
6.  **ОБЯЗАТЕЛЬНО** найдите своего созданного бота в Telegram и нажмите `/start`.
7.  В боте `@moonlighter_shop_bot` в главном меню скопируйте ваш **ID**.
8.  Вставьте ID в поле **"Telegram Chat ID"**.
    *   *Для уведомлений нескольким пользователям укажите ID через запятую: `12345,67890`.*
9.  Нажмите **"Test notification"**. Если сообщение пришло, нажмите **"Save"**.

> Через созданного бота можно управлять панелью командой `/start` (доступно только пользователям из Chat ID).

## Настройка Looter (Трейд-ссылки)

1.  Перейдите на вкладку **"Config"** -> **"Looter settings"**.
2.  Вставьте вашу трейд-ссылку (Trade URL) и нажмите **"Ok"**.
    *   Можно указать несколько ссылок через запятую.
    *   Пример: `https://steamcommunity.com/tradeoffer/new/?partner=...`

# Установка необходимого ПО и настройка системы

## Основные настройки

Настоятельно рекомендуется использовать оригинальный образ **Windows 10 Pro**. При использовании урезанных сборок могут возникать проблемы.

1. **Отключение игровых функций Windows** Отключите **Game Bar** и **Игровой режим** в настройках Windows.
2. **Отключение оверлеев**
   * **Nvidia:** Если установлено приложение Nvidia App, отключите Nvidia Overlay в настройках.
   * **Discord:** Если установлен Discord, **обязательно** отключите внутриигровой оверлей в настройках Discord, так как он конфликтует с панелью.
3. **Настройка файла подкачки (70 ГБ)** Необходимо увеличить размер файла подкачки до 70000 Мб.
   * Нажмите `Win + R`, введите `sysdm.cpl` и нажмите ОК.
   * Перейдите: "Дополнительно" -> "Параметры" (Быстродействие) -> "Дополнительно" -> "Изменить".
   * Выберите SSD диск. Нажмите "Указать размер".
   * Введите в оба поля (Исходный и Максимальный размер): `70000`.
   * Нажмите "Задать" и перезагрузите ПК.
4. **Очистка Steam userdata** Перейдите в папку `userdata` внутри директории Steam и удалите все находящиеся там папки.
5. **Имя пользователя Windows** Если имя папки пользователя (`C:\Users\Имя`) содержит русские буквы (кириллицу), необходимо изменить его на латинское (например, `Admin`, `FARM`). Это важно для кэширования шейдеров Nvidia.

## Настройка фарма

**Требования к драйверам (только Nvidia):**

* Проверьте версию драйвера в Панели Управления Nvidia ("Информация о системе").
* **Обязательно** должна быть установлена версия **546.65**.
* [Скачать драйвер 546.65](https://www.nvidia.com/download/driverResults.aspx/218047/en-us/)

**Подготовка игры:**

1. Запустите CS2 _без_ песочницы.
2. Зайдите на карту: `map de_inferno`.
3. После загрузки закройте игру.

### Запуск с использованием Avast

**Установка Avast:**

* [Скачать Avast Sandbox](https://mega.nz/file/Iqpy0Rha#JENOZ-OKbFAPQ68wyaK16M4KH5HsQhjKVT0Pz9XdGWE)
* При установке уберите все галочки, **КРОМЕ Песочницы (Sandbox)**.
* Активируйте файлом лицензии: [Скачать лицензию](https://mega.nz/file/su4FDbTI#NWJD6JWltpOEGWu9YjQzTjZwORXoLar5bUgcvFssYcw) (открыть двойным кликом).

**Настройка запуска Steam:**

1. Найдите файл `steam.exe` в папке Steam.
2. Нажмите ПКМ -> Свойства -> Совместимость -> Поставьте галочку **"Запускать эту программу от имени администратора"**.
3. Нажмите ПКМ на `steam.exe` и выберите пункт **"Всегда запускать в песочнице"** (Always run in sandbox).

### Запуск без использования Avast

Если раньше использовали Avast Sandbox убедитесь, что на steam.exe не стоит галочка **"Всегда запускать в песочнице"**

В настройках панели нажать чекбокс ![](<../.gitbook/assets/image (1).png>)

После этого можно приступать к фарму
